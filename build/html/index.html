About

Tijdens mijn opleiding Communicatie en Multimedia Design heb ik mij gespecialiseerd als Front End Developer.

Als Front End Developer ben ik natuurlijk constant bezig met HTML, CSS (of terwijl SASS, LESS en Stylus ;)) en JavaScript. Het afgelopen jaar heb ik hiernaast ook nog Node opgepakt. Tijdens stages ben ik ook nog bezig geweest met meehelpen met Angular. Hierdoor ben ik een brede generalist geworden die aan alle kanten wel iets weet.

De huidige Front Ender kan natuurlijk ook niet zonder tools. In mijn ontwikkelingsomgeving maak ik gebruik van Gulp en Git. Daarnaast heb ik een grote interesse in Progressive Web Apps en perfomance.
